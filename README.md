## IDF Build

#### Tool Chains

[ESP-IDF v5.5.1](https://docs.espressif.com/projects/esp-idf/en/v5.5.1/esp32s3/index.html)

#### Clone Submodules


```bash
git submodule update --init --recursive
```

#### Build

```bash
idf.py build
```

#### Flash

```bash
idf.py flash
```
